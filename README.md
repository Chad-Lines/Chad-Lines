## Open Source

### RPG Atlas

Created by [Driftwood Gaming](https://github.com/DriftwoodGaming), [RPG Atlas](https://github.com/DriftwoodGaming/RPGAtlas) is an open-source RPG engine focused on providing reusable systems for RPG development.

As a contributor, I implemented a full-featured quest framework including:

- Quest authoring tools
- Objective tracking
- Rewards and progression
- Branching outcomes
- Failure states
- Player journal integration

The contribution was accepted and merged into the project's primary codebase.
